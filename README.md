# 说明

此为作者平时工作经验总结


# 安装
    
    git clone https://github.com/nanlong/flask_zeus.git
    cd flask_zeus
    python setup.py install
            
    
或者
    
    
    pip install flask-zeus

# 依赖

1. python >= 3.5
2. flask >= 0.11
3. postgresql >= 9.5
4. 一些flask扩展 (flask-login, flask-wtf, flask-sqlalchemy, flask-restful, WTForms-JSON, ...等等)
